### Nominative
by adding -s / -ah
- a+s>o
- a+ah > **ā, h dropped**
- for other stems, **ah> o**

|     | singular |    plural     | neuter sl | nt plural |
| --- |:--------:|:-------------:| --------- | --------- |
| -a  |  a+s>o   |  ā {a+ah>ā}   | aṃ        | ā, āni    |
| -i  |    i     | ī, i>e>ay=ayo |           |           |
| -ī  |    ī     | ī, ino {ī<in} |           |           |
| -u  |    u     |  u, u>av=avo  |           |           |
| -ū  |    ū     | ū, uno {ū<un} |           |           |

## vocative
it is the same as nominative, only difference is:
- -a sg: o > a
- -ī sg: ī > i
- -ū sg: ū > u
